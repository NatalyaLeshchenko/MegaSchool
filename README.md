Система была создана для проведения технических интервью с кандидатами с помощью нескольких агентов, имитируя поведение реального рекрутера. Основные задачи системы: ведение диалога, борьба с отклонением кандидата от темы, анализ ответов кандидата, отслеживание тем, фиксирование финального результата в json-файле.
